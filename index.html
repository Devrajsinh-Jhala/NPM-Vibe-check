honest security boundary

A strong checkpoint, not a magical shield.

npx-vibe is a pre-execution risk scanner—not a sandbox, antivirus product, or formal audit. It inspects bounded package content and can miss runtime-only, dependency, native-code, or remote-response behavior. Proceed is useful context, never proof of safety.

bring your own model

No AI required. Your choice when useful.

Stay deterministic, keep review local with Ollama, or explicitly use OpenAI, Anthropic, Gemini, OpenRouter, Groq, Together, or a compatible endpoint.

privacy boundary

Your workspace is not the prompt.

Online review receives only bounded npm package metadata, findings, install scripts, and selected tarball files. Local project files, tokens, shell history, and environment-variable values stay out.

Built for developers who like fast tools and inspectable decisions.
