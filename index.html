01

Registry heuristics

Checks package age, version age, weekly downloads, maintainers, license, publish time, and npm update history.

02

Tarball inspection

Downloads the npm tarball without executing package code, verifies integrity, and inspects suspicious install-time files.

03

Rich package profile

Shows repository, GitHub stars, latest commit, last push, publisher, and maintainer context so humans can judge faster.

04

AI when useful

Runs static checks first, then uses Ollama or provider-agnostic online AI only when heuristic triggers make review worthwhile.

under the hood

Six small steps before trust.

  1. Resolve exact npm version
  2. Fetch registry + downloads
  3. Verify and unpack tarball safely
  4. Inspect hooks and selected source files
  5. Ask AI only when risk signals trigger it
  6. Proceed, caution, or block execution

privacy boundary

Your project stays yours.

npx-vibe never sends your local project files, npm tokens, shell history, or environment variables to a model. AI review receives only bounded package metadata, deterministic findings, install scripts, and selected files from the downloaded npm tarball.

Built for developers who like speed, but not surprise install scripts.
