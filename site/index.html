Why it exists

A small pause before a high-trust command.

npx is convenient because it removes friction. That same convenience also means a package can reach install hooks or executable code before you have inspected it. npx-vibe adds a quick pre-execution checkpoint for packages you do not fully know yet.

It is not a sandbox or antivirus product. It is a practical risk report: registry context, verified package content, install-time behavior, and the evidence behind each finding.

How it works

Deterministic by default. Explainable when it matters.

01

Resolve and verify

Resolve the exact npm version, fetch the tarball without executing it, and verify integrity metadata.

02

Inspect install behavior

Review lifecycle hooks and selected files for shell, network, secrets, obfuscation, and persistence signals.

03

Show useful evidence

Print file names, line-level excerpts, and a clear recommendation: Proceed, Caution, or Block.

Output

Readable terminal reports.

The output is designed to be useful in the moment: what package was inspected, what was found, why it matters, and what action is recommended.

Optional AI

Use AI only when you ask for it.

The default scan is heuristic-only and ignores ambient provider keys. If you want model interpretation, explicitly request local Ollama or an online provider. npx-vibe prints the resolved provider and model so the review is transparent.

Local Ollama for offline review
Online OpenAI, Gemini, Anthropic, and more
Profiles Fast, balanced, or strong model choices

Install

Start with one read-only check.

This command inspects the package without running its install scripts. Use it before executing unfamiliar npm packages.

Check a package npx npx-vibe --check esbuild
Install globally npm install -g npx-vibe
List AI models npx npx-vibe --models

Release assurance

Built for everyday developer use.

Cross-platform CI

Checks run across Node versions and operating systems before release.

Packed CLI smoke test

The published tarball path is tested, not only the source tree.

Privacy-conscious scope

Online AI review receives bounded package files, not your project workspace.

Security boundary

npx-vibe is a pre-execution scanner. It can help catch risky signals early, but it is not a sandbox, antivirus product, or formal audit. A Proceed verdict is useful context, not proof of safety.

Built for developers who want fast tools with inspectable decisions.
